Software Engineering Bootcamp

May 2024 - August 2024 (Expected)

15-week, immersive software engineering bootcamp. We focused first on front-end development and creating front-end web applications using HTML, CSS, and JavaScript and leverage ReactJS to build fully functional web applications. Leveraged the latest tools in artificial intelligence to test and debug. From there, we focused on back-end development and database storage with Python and SQL, ultimately leveraging Flask to create our own JSON API with React frontends to build slick, functional and interactive applications. Our culminating independent projects leveraged course materials and new exploration.
